# ⚡ DevPlatform — Frontend Workspace

This is the frontend client for **DevPlatform**, built using React 19, Next.js 16 (App Router with Turbopack), and Tailwind CSS v4.

---

## 🎨 Design Features

*   **Premium Dark UI**: Built with custom radial gradients, glassmorphism, and responsive layouts.
*   **Framer Motion Animations**: Features smooth viewport-reveal animations on scroll, hover transformations, and active tab transitions.
*   **Simulated Dashboards**: Mock data views for Repositories, AI Explainer, Kanban, and Weekly Charts directly on the landing page.

---

## 🚀 Running Locally

To run the development server, run the following command from this directory:

```bash
npm run dev
```

Alternatively, you can run it from the monorepo root:

```bash
npm run dev:frontend
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

---

## 🛠️ Tech Stack & Libraries

*   **Next.js 16** (App Router)
*   **React 19**
*   **Tailwind CSS v4** & PostCSS
*   **Framer Motion** (Animations)
*   **Zustand** (Auth State Management)
*   **Axios** (API Requests)
*   **Socket.io-client** (Real-time collaboration)
